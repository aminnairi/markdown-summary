# @aminnairi/markdown-summary

Command line interface for creating summary for your Markdown files.

## Usage

### Requirements

- Node
- NPX

### Usage

```bash
npx @aminnairi/markdown-summary README.md
```

### Requirements

- Docker
- Docker Compose


